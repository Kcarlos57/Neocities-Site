About The Effect

The circular inversion effect you're experiencing is created using CSS backdrop filters combined with JavaScript scroll detection. Each circle follows your scroll position and inverts the colors of everything behind it.

This creates a dynamic, interactive visual experience where the content literally changes color as you navigate through the page. The circles move at different speeds and directions, creating parallax-like motion.

The effect works on any content - text, images, backgrounds, and other elements all get inverted within the circular boundaries.
